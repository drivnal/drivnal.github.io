Simple snapshot backups
DEMO INSTALL
• Continuous incremental snapshots

All snapshot's copy only the changed data and hard link unchanged data from the previous snapshot. Allowing continuous incremental snapshots without the need to repeat full backups.

• Web interface

All configuration and backup/restore operations are done trough a simple web interface.

• Free and open source

Drivnal is free and open sourced under the Affero General Public License v3.

UBUNTU INSTALL

          $ sudo add-apt-repository ppa:drivnal/ppa
          $ sudo apt-get update
          $ sudo apt-get install drivnal
          Open http://localhost:6500/

ARCH LINUX INSTALL

          $ yaourt -S drivnal
          $ systemctl start drivnal.service
          $ systemctl enable drivnal.service
          Open http://localhost:6500/

PIP INSTALL

          $ sudo apt-get install python-pip
          $ sudo pip install drivnal
          $ sudo start drivnal
          Open http://localhost:6500/

DEVELOPMENT SETUP

          $ git clone https://github.com/drivnal/drivnal.git
          $ cd drivnal
          $ python2 server.py
          Open http://localhost:6500/

VAGRANT SETUP

          $ git clone https://github.com/drivnal/drivnal.git
          $ cd drivnal
          $ vagrant up testing
          $ vagrant ssh testing
          $ sudo drivnal
          Open http://localhost:6500/
          Open http://localhost:8080/collectd
