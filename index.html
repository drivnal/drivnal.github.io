Simple snapshot backups
DEMO INSTALL

Features

  • Continuous snapshots

    Snapshot's copy only the changed data. Unchanged data is hard link from the previous snapshot.

  • Web interface

    All configuration and backup/restore operations are done trough a simple web interface.

  • Email alerts

    Optional email alerts can be enabled to send an email when a scheduled snapshot has an error.

Install

Select linux distribution

1
Install "add-apt-repository" 
apt-get install -y python-software-properties
2
Add pritunl repository 
add-apt-repository ppa:drivnal/ppa
3
Update package list 
apt-get update
4
Install pritunl 
apt-get install -y drivnal
5
Use 
apt-get upgrade -y
to update when a new version is released
1
Download 
curl -O https://aur.archlinux.org/packages/pa/package-query/package-query.tar.gz
2
Download 
curl -O https://aur.archlinux.org/packages/ya/yaourt/yaourt.tar.gz
3
Extract archive 
tar zxf package-query.tar.gz
4
Extract archive 
tar zxf yaourt.tar.gz
5
Change directory 
cd package-query
6
Synchronize package databases 
pacman -Sy
7
Build and install package-query 
makepkg -si --asroot
8
Change directory 
cd ../yaourt
9
Build and install yaourt 
makepkg -si --asroot
10
Install drivnal 
yaourt -S drivnal
11
Start drivnal service 
systemctl start drivnal.service
12
Enable drivnal service to autostart 
systemctl enable drivnal.service
13
Use 
yaourt -Syua
to update when a new version is released
1
Install rsync package 
yum install rsync
2
Install python easy_install 
yum install python-setuptools
3
Install python-pip 
easy_install pip
4
Install importlib if python2.7 is not installed 
pip install importlib
5
Install drivnal 
pip install drivnal
6
Start drivnal service 
start drivnal
7
Use 
pip install --upgrade drivnal
to update when a new version is released. When upgrading with pip 
/etc/drivnal.conf
will be overwritten so a backup will need to be created before upgrading

Configure drivnal

1
Goto the servers IP address on port 6500 
http://<SERVER_ADDRESS>:6500/
2
In the new backup volume section enter a name for the backup volume
3
Select a directory to backup and an empty directory to store the backups
4
Select any directories to exclude them from the backup
5
Click "New Snapshot" to create the first snapshot.

Restore data

1
To restore data first select a snapshot from the snapshot list
2
Select the files and directories that need to be restored then drag them to right
3
Restoring files will overwrite files with the same name
4
Wait for the restore task to complete before accessing the data
